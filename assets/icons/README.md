# App Icon

Place the app icon here:

## Requirements
- **Filename**: `app_icon.png`
- **Size**: 1024x1024 pixels
- **Format**: PNG
- **No alpha/transparency**: iOS requires no transparency

## Generating the icon

After placing the icon file, run:
```bash
flutter pub get
dart run flutter_launcher_icons
```

This will generate all required iOS icon sizes automatically.
